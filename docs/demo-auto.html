Evipedia Widget — Automatic Mode

Loads the real widget.js against live evipedia.ai data in auto mode. Nothing on this page is marked up — the widget scans the text itself and highlights every review name it recognizes.

→ See also the manual mode demo.

The evidence base for supplements varies widely. Some people take creatine for strength and muscle mass, use melatonin to shift sleep timing, or reach for magnesium for cramps. In cardiometabolic health, metformin and statins are frequently discussed, and riboflavin has been trialed for migraine prophylaxis.

Auto mode is deliberately non-intrusive elsewhere: links like creatine, melatonin inside code, and form fields like are all left untouched.

Note. Cards render in a Shadow DOM, isolated from this page's styles. Only terms that resolve against reviews.json get the dotted underline; unknown or unmarked terms are left alone.
